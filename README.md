## 🧠 핵심 역량

- **성능 최적화** · `측정 → 병목 규명 → 가설 검증` 루프로 Konva.js의 렌더링 병목(나이브 AABB `O(N)`)을 진단하고, `Uniform Grid` 공간 해싱을 직접 적용해 탐색 복잡도를 `O(N) → 평균 O(1)`로 낮춰 `780ms → 12ms`(98%↓)·60fps를 달성했습니다.
- **FE 아키텍처 설계** · 라이브러리 제약(VanillaJS) 환경에서 단일 Store(SSoT) + 단방향 `Pub-Sub` EventBus를 직접 설계해, 기능 추가 시 수정 파일을 `4~5개 → 1곳`으로 줄였습니다.
- **유저 중심 기획** · 400여 명 실무자 VoC를 `ICE Score`로 우선순위화해, 6주 만에 실제 업무에 투입된 사내 서비스로 구체화했습니다.
- **소통과 리더십** · 2년간의 강사 경험과 팀장 13회(학부 10 / SSAFY 3)를 기반으로, 고객인 비개발 직군과의 이해관계를 조율했습니다.
- **알고리즘** · 백준 `Diamond V`(상위 1%), 400일+ 연속 풀이 — 알고리즘을 실무 최적화의 도구이자 기반으로 활용합니다.
- **AI 적용** · LangChain 기반 `RAG`·캡셔닝 멀티모달 파이프라인을 설계하고, `Vibe Coding`으로 확보한 공수를 서비스 안정화에 재투자합니다.
